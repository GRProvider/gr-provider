# GR Provider Bot - Полный Контекст

## ПРОЕКТ: gr_provider (GR Navigator)

### ОБЩЕ ОПИСАНИЕ

**GR Provider** — это сервис корпоративной разведки и government relations для компаний на украинском рынке.

Текущий фокус: HealthTech (longevity/rehabilitation/sensors) + психоделическая терапия (ПТСР у ветеранов).

**Структура бизнеса:**
- Добывающая часть: OSINT (открытые источники)
- Продающая часть: аналитика + GR-услуги
- Монетизация: подписки от партнеров + revenue share от deals + консультации

## ЗАДАЧА: СОЗДАТЬ TELEGRAM БОТ

### ЧТО ДОЛЖЕН ДЕЛАТЬ БОТ

**Основная функция:**
Пользователь пишет вопрос в Telegram → бот анализирует → возвращает релевантный документ с маршрутом.

**Сценарии:**
1. User: "Як отримати ліцензію для HealthTech?"
   Bot: Маршрут + сроки + ризики + "Как сприяти" + CTA

2. User: "Как зарегистрировать психоделическую терапию?"
   Bot: (на русском) маршрут Перекласифікації + сроки + ризики + CTA

3. User: "How to get a medical device license?"
   Bot: (на англ) маршрут ліцензії HealthTech + все детали + CTA

### МЕХАНИКА РАБОТЫ БОТ

**Не AI-based (пока без Claude API):**
- Простой matching по keywords/commonQuestions
- Если найден документ → вернуть всю информацию
- Если не найден → предложить "Напиши @gr_provider для консультації"

**Тип бота:**
- Polling mode (бот сам запрашивает обновления у Telegram)
- ИЛИ Webhook mode (Telegram сам отправляет)

**Для какого чата работает:**
- Личные сообщения боту (@gr_navigator_bot)
- НЕ в групах (пока)

### ОТВЕТ БОТА (FORMAT)
```
Документ: Ліцензія HealthTech (МОЗ)

Сроки:
- Мінімум: 20 днів
- Реалістично: 30 днів  
- Максимум: 45 днів

Маршрут:
1. Підготовка документів (5-7 днів)
2. Подача в МОЗ (1 день)
3. Технічна перевірка (5-7 днів) [поточний етап]
4. Прийняття рішення (7-10 днів)
5. Видача ліцензії (2-3 дні)

Орган видачі: Міністерство охорони здоров'я

Типові ризики:
- Неповний пакет документів
- Політична волокита (штучне замораживання документа)
- Вимога додаткових документів
- Конфлікт інтересів у комісії

Як сприяти:
- GR Provider: перевірка пакету документів
- GR Provider: запит про статус через 7 днів
- GR Provider: зустріч із інспектором (підготовка)
- GR Provider: контакти в МОЗ та навігація

Потрібна допомога? Напиши @gr_provider для консультації
```
